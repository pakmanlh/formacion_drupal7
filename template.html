Qué es Drupal?

  • Creado por Dries Buytaert en 1999
  • CMS escrito en PHP
  • Open Source
  • Matenido por una enorme comunidad (1,053,901 usuarios en drupal.org)
    • 26,385 Módulos
    • 1,980 Temas
    • 796 Distribuciones
    • 32,782 Desarrolladores
  • Security Team
  • Versión 7 estable
  • Versión 8 en desarrollo

Requisitos técnicos Drupal

  • Web Server: Apache or Microsoft IIS
  • PHP: 5.2 or higher
  • Database Server: MySQL - 5.0 or higher, PostgreSQL - 8.3 or higher, or SQLite

Instalación de Drupal

  • Descargar Drupal https://drupal.org/project/drupal
  • Crear una BBDD nueva
  • Crear settings.php y darle permisos de escritura
  • Crear carpeta «files» y darle permisos de escritura
  • Acceder a la ruta y seguir la instalación.
  • Importante! Cambiar permisos a settings.php

Views

  • Creado por Earl Miles (Merlin of Chaos)
  • Creado para listar contenido (de cualquier tipo)
  • Sin necesidad de saber SQL
  • Concepto «Vista» (Views) de contenido
  • Una Vista dispone de diferentes «displays» o visualizaciones
  • Los Displays pueden ser Bloques, páginas, etc
  • Activar Views UI :)
  • Exportables / Manejables como Arrays
  • Demo!

Webform

  • Permite crear «entidades» con formularios
  • Permite envío de Datos a través de correo
  • Guarda los datos y permite administrarlos / exportarlos
  • Acciones condicionales (mostrar / ocultar campos)
  • Mensajes de confirmación, límite de registros
  • Exponer formularios como bloques
  • Módulos como Form Builder o Options element mejoran la UI de creación.
  • Demo!
